# student-analysis
En mall för elever att träna på pandas och Jupyter Notebook.

## Syfte
Denna repository innehåller:
- `student_analysis.ipynb`: huvudsaklig arbetsbok där du gör analyserna
- `student_data.csv`: exempeldata
- `generate_data.py`: script för att skapa/generera data (valfritt)

Elever får använda AI (till exempel Copilot) i arbetet.

## Börja använda mallen
1. Skapa ett nytt repository med denna mall: `Use this template`.
2. Installera de bibliotek som behövs:
`pip install pandas numpy matplotlib`

## Lös uppgifterna
1. Läs uppgifterna i notebooken och följ instruktioner i cellerna.
2. Ha ett experimentellt förhållningssätt och se om du kan komma på något annat intressant att göra med datan.

## Kontrollera löst
När du är klar:
- Se till att notebooken körs utan fel från början till slut.
- Spara ditt arbete med Git och lämna in som länk till ditt repository.
